abstractqueryrewriter:_extend_query_with_source_schema(query, source_schema_id) line 75
Add the source database schema to all query elements that represent a table. Table elements in the query structure are lacking the information which schema they belong too. But without this information, the database cannot locate the table, because the table name is only valid in the context of the containing schema. So this method adds the missing information by adding the source schema of the Virtual Schema into the table elements.

Parameters:

  • query
  • source_schema_id

Returns:

    query with table elements that contain the source schema
